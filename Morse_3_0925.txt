Paid License - Licence that is available to everyone, but only if you pay the proprietor money.

Shareware - Software that is available to be shared for free

Trialware - Software that can be tryed for free for a limited amount of time. When the time runs out, you will either not have access to thee software anymore,
or you will have to pay to keep it

Ad-ware - Software that brings up ads without user permission

Crippleware -  Software that is sold like fully functional programs, but is not fully functional

Freemium - Software that is technically free, but offers other parts of the software for money

Freeware - Software that is free
